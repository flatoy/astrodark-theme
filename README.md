# Astrodark Theme

Astrodark is a dark theme for Obsidian, inspired by the [Astronvim](https://astronvim.com/) dark theme/One Dark Pro. It aims to provide a clean, high-contrast coding and writing environment with a modern aesthetic.

## Fonts

This theme uses **Iosevka Nerd Font** for all text elements, providing a consistent monospace look throughout the interface.

- **Interface & Text:** `Iosevka Nerd Font`, `monospace`

## Color Palette

The color scheme is built around a core set of dark backgrounds and vibrant accents:

### Backgrounds
- **Main Editor:** `#282c34`
- **Sidebar / Gutters:** `#21252b`
- **Darker Borders:** `#181a1f`

### Foreground
- **Main Text:** `#E6E6E6` (Bright Light Grey)
- **Muted Text:** `#ABB2BF`

### Accents
- **Red:** `#E06C75`
- **Orange:** `#D19A66`
- **Yellow:** `#E5C07B`
- **Green:** `#98C379`
- **Cyan:** `#56B6C2`
- **Blue:** `#61AFEF`
- **Purple:** `#C678DD`
- **Pink:** `#FF79C6`

## Features
- **Custom File Icons:** Uses Nerd Font icons for folders and file types.
- **Syntax Highlighting:** Matches the One Dark Pro scheme.
- **UI Polish:** Custom checkboxes, active tab styling, and clean status bar.
